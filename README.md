# Associations Between Echocardiographic Traits and AI-ECG Predictions of Heart Failure

This repository hosts the analysis code for the article "Associations Between Echocardiographic Traits and AI-ECG Predictions of Heart Failure".

`main.ipynb` contains the primary internal-cohort analysis linking AI-ECG heart-failure prediction scores to echocardiographic traits.

`echo-next.ipynb` runs inference with the AI-ECG heart-failure model described in ["Deep learning for heart failure detection in the electrocardiogram"](https://www.nature.com/articles/s41746-026-02774-4#citeas) on the [EchoNext dataset](https://physionet.org/content/echonext/1.1.1/) and reproduces the external echocardiographic association analyses.
